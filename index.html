Kommando Bereich

VS Inhalte bei Github hochladen

1. Schritt: ein neues Repository erstellen. Dafür oben rechts das Profil öffnen und click auf "your repositories".

Dann wird über den Button "new" ein neues Repository erstellt.

Jetzt wechselst du zurück zu Visual Studio Code.
Hier öffnest du das betreffende Dokument und rufst das Terminal auf. Du benötigst nun folgende drei Anweisungen:

  • git init
  • git add .
  • git commit -m "Projektname XY"

Gib jede Zeile einzeln ein, bestätige mit Enter und gib die nächste Zeile ein. Bei der dritten Zeile kommt der individuelle Projektname in die Anführungszeichen.

Im nächsten Schritt musst du 3 Zeilen aus Git hub bei VS einfügen. Die betreffenden Zeilen findest du am Ende des Erstellungsprozesses deines neuen Repositories. Für die Verlinkung werden dir vier Möglichkeiten gegeben, von denen die dritte lautet: (hier für das Beispieldokument namens box model-2_1)

Kopiere diese drei Zeilen aus Github in die Zwischenablage. Dies geht auch anhand des Doppelkästchen-Icons rechts neben dem Code. Im Anschluss setzt zu sie im VS Terminal wieder ein. Das sollte so aussehen. Die drei Zeilen aus Git hub sind weiß hinterlegt.Starte den Vorgang wieder mit der Enter-Taste.

Im Anschluss sollte VS dir einige Zeilen Ausgeben. Der Umfang kann hier variieren, da er von der Datei abhängt. In dieser Beispielabbildung ist die Ausgabe nicht sehr umfangreich.
